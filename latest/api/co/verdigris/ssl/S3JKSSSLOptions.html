class S3JKSSSLOptions extends SSLOptions

Uses Java KeyStore file stored on Amazon Web Services S3 to build the SSL context and handler which is used by the Cassandra driver to enable client-to-node encryption.

Can be instantiated directly with S3 URL and the Java KeyStore password or through environment variables which is useful if accessing from AWS Lambda.
